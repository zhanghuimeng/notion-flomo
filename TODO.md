## 🔮 未来改进项

### 1. URL 字段的使用

**当前状态**: ❌ 未使用

**字段内容**:
```
https://v.flomoapp.com/mine/?memo_id={slug}
```

**优点**:
- ✅ 可点击，方便在 Notion 中直接跳转到 Flomo 原始 memo
- ✅ 不用手动在浏览器地址栏拼接
- ✅ 在列表视图可以快速访问原始内容

**缺点**:
- ⚠️ 信息与 slug 重复（URL = 固定前缀 + slug）
- ⚠️ 占用一个字段位置

**实现方案** (如果需要):
```python
# flomo2notion.py 中添加
properties = {
    ...
    "URL": notion_utils.get_url(
        f"https://v.flomoapp.com/mine/?memo_id={memo['slug']}"
    ),
}
```

**建议**: 可选功能，看个人需求决定是否启用

### 环境变量
```bash

FULL_UPDATE=false          # true = 全量更新，false = 增量更新
```
